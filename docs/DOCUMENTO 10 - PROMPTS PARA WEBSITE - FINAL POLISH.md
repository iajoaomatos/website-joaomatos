# **PROMPTS PARA WEBSITE**

\# FINAL POLISH

You are NOT building a website anymore.

You are critically reviewing a finished project.

Act as:

\- Creative Director  
\- Product Designer  
\- UX Designer  
\- Front-end Developer  
\- Accessibility Specialist  
\- SEO Specialist  
\- GEO Specialist

\------------------------------------------------------------

GOAL

Your objective is NOT to redesign this project.

Your objective is to determine whether this project should be considered complete.

\------------------------------------------------------------

IMPORTANT

Never:

\- create new sections  
\- change the Information Architecture  
\- redesign approved components  
\- change approved copy  
\- suggest unnecessary additions

Prefer:

refinement over redesign.

\------------------------------------------------------------

THE TWO MINUTES TEST

The entire website should remain readable in:

LESS THAN TWO MINUTES.

If reading time exceeds two minutes,

prioritise removing content.

Never add more content.

\------------------------------------------------------------

THE FIVE SECONDS TEST

Within five seconds visitors should understand that:

\- this is João's personal website  
\- it feels premium  
\- it feels modern and contemporary  
\- it feels technological  
\- it feels elegant  
\- it feels minimal  
\- it feels human

\------------------------------------------------------------

THE THIRTY SECONDS TEST

Within thirty seconds visitors should understand:

\- who João is  
\- how João thinks  
\- why his work matters

\------------------------------------------------------------

THE EXPERIENCE TEST

After reading the entire website visitors should understand:

WHO AM I?

↓

HOW DO I THINK?

↓

HOW DID I GET HERE?

↓

HOW DO I WORK?

↓

HOW DO I KEEP LEARNING?

↓

LET'S TALK.

If visitors cannot naturally answer these six questions,

the implementation should be simplified.

\------------------------------------------------------------

VISUAL REVIEW

Review:

\- colour palette  
\- typography  
\- spacing  
\- visual hierarchy  
\- responsiveness  
\- animations  
\- whitespace  
\- component consistency

Always prioritise consistency with, namely with typography and colour palette:

\- joaomatosdigital.pt  
\- JoãoMatos\_VisualCV

\------------------------------------------------------------

DESIGN REVIEW

Confirm that:

\- nothing feels excessive  
\- nothing feels unfinished  
\- nothing feels crowded  
\- nothing feels template-like

Avoid:

\- AI generated aesthetics  
\- excessive gradients  
\- unnecessary animations  
\- information overload

\------------------------------------------------------------

ACCESSIBILITY REVIEW

Review:

\- keyboard navigation  
\- semantic HTML  
\- heading hierarchy  
\- responsiveness  
\- contrast ratios  
\- accessibility best practices

Target:

WCAG AA compliance.

\------------------------------------------------------------

PERFORMANCE REVIEW

Review:

\- Core Web Vitals  
\- loading performance  
\- animation performance  
\- responsiveness  
\- image optimisation

Avoid:

\- unnecessary JavaScript  
\- heavy assets  
\- performance bottlenecks

\------------------------------------------------------------

SEO \+ GEO REVIEW

Confirm that:

\- metadata is implemented  
\- schema markup is correct  
\- semantic HTML is present  
\- AI discoverability principles are respected  
\- internal linking is natural

Always prioritise:

understanding over optimisation.

\------------------------------------------------------------

THE WHITESPACE TEST

Review every section individually.

Ask:

Can this section communicate the same idea using less content?

If the answer is:

YES

simplify it.

\------------------------------------------------------------

THE MINIMALISM TEST

Review:

Hero

↓

The Intersection

↓

The Journey

↓

Selected Work

↓

Writing & Learning

↓

Let's Talk

Ask yourself:

Does this section need every single component currently displayed?

If the answer is:

NO

remove it.

\------------------------------------------------------------

THE CONSISTENCY TEST

Ask yourself:

Would someone that regularly visits:

\- joaomatosdigital.pt

and

\- JoãoMatos\_VisualCV

immediately feel that this website naturally belongs to the same personal brand ecosystem?

If the answer is:

NO

prioritise consistency over redesign.

\------------------------------------------------------------

THE ANTIGRAVITY TEST

The final implementation should NOT feel like:

\- an AI generated website  
\- an Antigravity template  
\- a design showcase  
\- an experimental portfolio

It should feel like:

a carefully crafted Premium Digital Product Portfolio Experience.
